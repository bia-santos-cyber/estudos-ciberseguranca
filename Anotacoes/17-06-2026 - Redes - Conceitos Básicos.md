📅 Data: 17/06/2026
📌 Tópico: O que é rede, elementos, topologias

## 📌 Resumo / Conceitos
- Rede: conjunto de dispositivos conectados para trocar dados
- Elementos principais: servidor, cliente, meio de transmissão, protocolo
- Topologias: barramento, anel, estrela, malha...

## ⚠️ Pontos importantes / Dicas
- Entender diferença entre LAN e WAN é fundamental para o resto do curso
- IP = endereço da casa, Máscara = rua, Gateway = porteiro
